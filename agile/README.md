# Scrum・Agile資格クイズ

SMC / PSM I / PSM II / CSPO / SAFe Agilist 対応のクイズサイトです。

## 仕様

- 5カテゴリ対応
- 各カテゴリ180問
- 表示は50問ランダム
- 問題文重複を除外して出題
- `window.quizData.smc` のようにJSから読み込み
- 回答後、自動で次の問題へ移動
- 左右にAdSense広告枠あり
- PC / スマホ対応

## URL例

```text
index.html
index.html?type=smc
index.html?type=psm1
index.html?type=psm2
index.html?type=cspo
index.html?type=safeAgilist
```
